# MagickaBBS

A Bulletin Board System for UN*X like platforms.

For documentation, see [https://magickabbs.com/docs/](https://magickabbs.com/docs/)

Update February 2026:
Code has been updated to compile under debian tixie x86_64 with more recent gcc and libraries.  If you attempt to compile under another os, you may need to make similar changes.  Unsupported and placed here for posterity.

Update March 2026:
Updated to use libcdk debian package instead of compiling its own.  libcdk dependencies added to install.txt.  Other OSes may still require compiling so the source is still included.
